InputTags

Useful for getting input and showing contextual Tags aside.

TraefikCompression
<InputTags
  placeholder="Input Tags Placeholder"
  tags={['Traefik', 'Compression']}
  onChange={value => alert(`onChange called with '${value}'`)}
/>

Props

PropTypeDescription
valuestringOptional. The initial value for the input field.
placeholderstringOptional. Placeholder for the input field.
tagsstring[]Optional. List of tags to show next to the Input field.
optionsstring[]Optional. List of options to show as autocomplete.
maxInlineTagsnumberOptional. Maximum number of tags to show inline next to the input field, aditional tags will be shown below the field. There's no overall limit for the quantity of tags.
renderTag(tag: string, onDeleteTag?: (tag: string) => void) => ReactNodeOptional. Custom function to be used to render the Tags.
renderOption(tag: string, onClick: (value: string) => void) => ReactNodeOptional. Custom function to be used to render the options in the list for autocomplete.
onInputChange(value: string) => voidOptional. Function that is triggered after each input change. Triggers after each key press.
onChange(value: string) => voidOptional. Function that is triggered when is pressed the Enter key on the input.
onDeleteTag(tag: string) => voidOptional. Function triggered after a close button is clicked on a Tag. It sends the deleted tag value as parameter.

Example

TraefikCompression
<InputTags
  placeholder="Input with autocomplete"
  tags={['Traefik', 'Compression']}
  options={['Redirect Path', 'Redirect Scheme']}
  onChange={value => alert(`Value submitted: '${value}'`)}
  onDeleteTag={tag => alert(`Tag deleted: '${tag}'`)}
/>
