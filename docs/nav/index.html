Nav

A main navigation bar

<Nav>
  <NavItem>
    <img src="https://containo.us/assets/img/traefik-logo.svg" height="30px" />
  </NavItem>
  <NavGroups>
    <NavGroup variant="right">
      <NavItem variant="active">Documentation</NavItem>
      <NavItem>Github</NavItem>
    </NavGroup>
    <NavGroup variant="left">
      <NavItem>Box</NavItem>
      <NavItem variant="active">Layout</NavItem>
      <NavItem>Button</NavItem>
    </NavGroup>
  </NavGroups>
</Nav>

NavGroup Props

PropTypeDescription
childrenReactNodeThe content to render
variantleft | rightThe position of children in the navigaion bar

NavItem Props

PropTypeDescription
childrenReactNodeThe content to render
variantactive | normalThe navigation state of this item
asa | buttonThe HTML element to use to render the navigation item

NavGroup Variant

There are 2 variants to choose from.

<Nav>
  <NavGroup variant="right">
    <NavItem>
      <Text size={2} textColor="grays.5" fontWeight={600}>
        Button on the right
      </Text>
    </NavItem>
  </NavGroup>
</Nav>
<Nav>
  <NavGroup variant="left">
    <NavItem>
      <Text size={2} textColor="grays.5" fontWeight={600}>
        Button on the left (default)
      </Text>
    </NavItem>
  </NavGroup>
</Nav>
<Nav>
  <NavGroup variant="right">

  </NavGroup>
