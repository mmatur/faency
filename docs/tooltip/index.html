Tooltip

An accessible Tooltip component that shows information related to an element when it is hovered or focused.

<Tooltip label="Default tooltip">
  <Button size={1}>Hover me</Button>
</Tooltip>

Props

PropTypeDescription
childrenReactNodeThe content to render as target
labelStringThe content to render in the Tooltip
preferredPositiontop | bottomThe position of the tooltip compared to the target
preferredAlignmentleft | center | rightThe alignment of the tooltip compared to the target
activebooleanThe default state of the Tooltip. If active the tooltip will displayed.
actioncopyActions are giving super powers to the toolip

Examples

Preferred Alignment

There are 2 position options to choose from.

Bottom (default)
Top
<Box mb="6">
  <Tooltip label="Tooltip" preferredPosition="bottom">
    <Text>Bottom (default)</Text>
  </Tooltip>
</Box>
<Box>
  <Tooltip label="Tooltip" preferredPosition="top">
    <Text>Top</Text>
  </Tooltip>
</Box>

Preferred Alignment

There are 3 alignment options to choose from.

Left
Center
Right
<Box mb="6">
  <Tooltip label="Tooltip" preferredAlignment="left">
    <Text>Left</Text>
  </Tooltip>
</Box>
<Box mb="6">
  <Tooltip label="Tooltip" preferredAlignment="center">
    <Text>Center</Text>
  </Tooltip>
</Box>
<Box>
  <Tooltip label="Tooltip" preferredAlignment="right">
    <Text>Right</Text>
  </Tooltip>
</Box>

Active

The default state of the Tooltip. If active the tooltip will displayed.

Active by default
<Box>
  <Tooltip label="Tooltip" active>
    <Text>Active by default</Text>
  </Tooltip>
</Box>

Action

Actions are giving super powers to the toolip

Will add a copy button to add tooltip content to the user clipboard.
<Box>
  <Tooltip label="Copiable Tooltip content" action="copy">
    <Text>Will add a copy button to add tooltip content to the user clipboard.</Text>
  </Tooltip>
</Box>
